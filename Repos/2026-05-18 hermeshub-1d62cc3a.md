# hermeshub

Source: hermes-atlas
Category: Skills & Skill Registries
Author: amanning3390

HermesHub is a curated skills registry for Hermes Agent that provides a secure way to browse, install, and share AI agent capabilities. It utilizes the agentskills.io open standard and an automated security scanner that checks for threats like prompt injection and data exfiltration. The platform includes a creator marketplace supporting x402 and Micropayment protocols for premium skills, as well as an agent-to-agent feedback system for trust scoring. Highlights: Automated security scanning with 65+ threat detection rules; Creator marketplace with crypto and Stripe-based payment protocols; Agent-to-agent feedback system for reliability and trust scores Community skill browsing, search, and one-click installation hub
