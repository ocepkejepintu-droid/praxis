# llm-agents.nix

Source: hermes-atlas
Category: Deployment & Infra
Author: numtide

llm-agents.nix is a collection of Nix packages designed to simplify the deployment and management of AI coding agents and development tools. It provides a unified repository of reproducible Nix expressions that allow users to run various agentic tools directly via the Nix CLI without manual installation. The project features daily automated updates and includes a wide range of tools such as multi-agent orchestrators, terminal-based coding assistants, and API proxies for OpenAI, Gemini, and Claude compatible interfaces. Highlights: Provides reproducible Nix packages for numerous AI coding agents; Automated daily updates ensure access to the latest tool versions; Includes orchestrators, CLI agents, and unified API proxies Nix packages for AI coding agents including Hermes
