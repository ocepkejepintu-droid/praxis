# hermes-blockchain-oracle

Source: hermes-atlas
Category: Integrations & Bridges
Author: gizdusum

Hermes Blockchain Oracle is a Model Context Protocol (MCP) server that provides Hermes Agent with real-time access to the Solana blockchain. It functions as a bridge between natural language queries and on-chain data by exposing specialized tools for wallet, token, and transaction analysis. The server enables users to monitor network health, track whale movements, and manage NFT portfolios through conversational interactions. By integrating directly with Solana RPC nodes, it eliminates the need for manual block explorer searches. Highlights: Provides real-time Solana wallet balances and token portfolio tracking; Detects large-scale whale transfers with configurable movement thresholds; Retrieves NFT metadata, collection details, and floor price information Solana blockchain intelligence MCP server — wallets, whales, tokens, NFTs, network health
