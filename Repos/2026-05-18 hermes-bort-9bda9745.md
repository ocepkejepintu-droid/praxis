# hermes-bort

Source: hermes-atlas
Category: Integrations & Bridges
Author: BORT-AGENTS

Hermes-bort is a plugin for Hermes Agent that enables interaction with BORT agent NFTs on the Binance Smart Chain mainnet. It allows Hermes processes to read agent states, execute on-chain actions via operator-signed writes, and manage portable session memory. The plugin integrates with the BORT logic contracts and knowledge registries to anchor chat history and evolution metrics to IPFS. This setup facilitates a persistent identity and memory for autonomous agents that can be transferred alongside the underlying NFT. Highlights: Reads multi-source agent state, logic snapshots, and marketplace data.; Executes operator-signed writes gated by local policy and approval prompts.; Anchors session memory to IPFS and on-chain knowledge registries.
