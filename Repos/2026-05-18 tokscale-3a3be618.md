# tokscale

Source: hermes-atlas
Category: Developer Tools
Author: junhoyeo

Tokscale is a CLI tool and visualization dashboard designed to track token consumption and costs across various AI coding agents. It works by scanning local databases and session files from supported clients to aggregate usage data into a native Rust TUI or a web-based 3D contribution graph. The tool provides detailed daily summaries and model-specific statistics for a wide range of integrations, including Hermes Agent, Claude Code, and GitHub Copilot CLI. Users can also submit their data to a global leaderboard to create public usage profiles. Highlights: Tracks token usage and costs across multiple AI coding agents; Features a native Rust TUI for local data visualization; Supports Hermes Agent, Claude Code, Gemini, and Cursor IDE CLI tool for tracking token usage from Claude Code, OpenClaw, Hermes, Codex, Gemini, and more
