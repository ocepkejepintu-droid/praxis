# mem0

Source: hermes-atlas
Category: Memory & Context
Author: mem0ai

Mem0 is a universal memory layer designed to provide AI agents and assistants with long-term context and personalization. It utilizes a multi-signal retrieval algorithm that fuses semantic search, BM25 keyword matching, and entity linking to recall user preferences and past interactions. The system supports temporal reasoning to distinguish between past events and current states while maintaining separate memory tiers for users, sessions, and agents. It is available as an Apache-2.0 self-hosted package, a Python/Node.js library, or a managed cloud service. Highlights: Multi-level memory for users, sessions, and autonomous agent states; Hybrid retrieval combining semantic, keyword, and entity-linked signals; Temporal reasoning for time-aware ranking of past and future events Universal memory layer for AI Agents — official Hermes Agent memory provider, available as managed cloud or Apache-2.0 self-hosted OSS
