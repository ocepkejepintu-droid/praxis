# SkillClaw

Source: hermes-atlas
Category: Skills & Skill Registries
Author: AMAP-ML

SkillClaw is an agentic evolution framework that manages and improves AI agent skill libraries through real-world interaction data. It operates a background loop that automatically deduplicates, merges, and refines skills across different sessions, devices, and users. The system enables a unified skill repository where experience gained by one agent instance—such as a frontend task—can be cross-pollinated to benefit other instances like backend or ML agents. It natively integrates with Hermes, OpenClaw, and various OpenAI-compatible APIs to provide continuous skill growth without manual intervention.
