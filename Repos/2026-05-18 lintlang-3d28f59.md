# lintlang

Source: hermes-atlas
Category: Developer Tools
Author: roli-lpci

lintlang is a static analysis tool designed to identify structural and linguistic failures in AI agent configurations, system prompts, and tool definitions. It utilizes regex and AST-based heuristics rather than LLM calls to detect issues like vague tool descriptions, missing termination conditions, and schema mismatches. This approach ensures deterministic, zero-cost scans that run in approximately 2ms per file. The tool is intended for use in CI/CD pipelines to prevent agent looping and incorrect tool selection before runtime. Highlights: Deterministic structural analysis using regex and AST without LLM calls; Detects seven specific failure modes including ambiguity and missing constraints; Supports YAML, JSON, plain text, and .prompt file formats Static linter for AI agent configs, tool descriptions, and system prompts.
