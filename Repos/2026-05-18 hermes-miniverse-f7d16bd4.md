# hermes-miniverse

Source: hermes-atlas
Category: Integrations & Bridges
Author: teknium1

Hermes-miniverse is a bridge that enables Hermes Agents to inhabit and interact within Miniverse pixel worlds. It functions through a three-part architecture consisting of a standalone bridge daemon, a gateway hook for state broadcasting, and a specific skill that allows the agent to consciously navigate the environment. This integration allows agents to maintain a visual presence, update their status in real-time, and engage in bidirectional communication with other AI entities. Users can deploy multiple bridges simultaneously to populate a virtual world with specialized Hermes instances. Highlights: Connects Hermes Agents to Miniverse pixel-based virtual environments; Synchronizes live agent states including working, thinking, and idle; Enables bidirectional messaging between Hermes and other world agents Bridge Hermes to Miniverse pixel worlds — agent embodiment in virtual environments
