# vessel-browser

Source: hermes-atlas
Category: Developer Tools
Author: unmodeled-tyler

Vessel is an open-source, Chromium-based browser designed specifically for autonomous agent control rather than traditional human-centric browsing. It provides a persistent runtime environment where agents drive workflows via the Model Context Protocol (MCP) while humans supervise through a visible UI. The browser maintains durable state across sessions, including annotated checkpoints, action undo capabilities, and agent-readable bookmarks with structured metadata. This setup eliminates the "black box" nature of headless automation by allowing users to audit, intervene, and redirect long-running agent tasks in real-time. Highlights: Native MCP control for external agent harnesses like Hermes Agent; Human-in-the-loop supervisory UI with live approvals and runtime controls; Automated page schema inference for structured entity and form detection AI-native browser built for autonomous agent control via MCP
