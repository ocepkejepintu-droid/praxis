# pan-ui

Source: hermes-atlas
Category: Workspaces & GUIs
Author: Euraika-Labs

Pan is a self-hosted web dashboard and workspace designed specifically for the Hermes Agent runtime. It provides a centralized interface to manage agent interactions through a chat-first environment that supports SSE streaming and tool timelines. Users can discover and install skills from skills.sh, manage MCP integrations, and organize work into isolated profiles with unique memory and sessions. The system includes a setup wizard that automatically detects Hermes configurations and can run as a background daemon or system service. Highlights: Chat-first workspace with real-time SSE streaming and tool approval cards; Integrated marketplace for discovering and installing skills and MCP servers; Isolated profile management for separate sessions, memory, and API keys Self-hosted AI workspace with chat, skills, extensions, memory, profiles, and runtime controls
