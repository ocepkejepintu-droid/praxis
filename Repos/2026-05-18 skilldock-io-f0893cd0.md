# skilldock.io

Source: hermes-atlas
Category: Skills & Skill Registries
Author: chigwell

SkillDock is a registry and SDK for managing reusable AI skills based on the AgentSkills specification. It provides a Python client and CLI that interact with an OpenAPI-driven backend to discover, install, and version agent capabilities. The platform supports a commerce layer for buying and selling skills using Stripe or TON blockchain payments. Developers can package local skill folders, manage dependencies, and publish immutable releases with metadata defined in Markdown frontmatter. Highlights: OpenAPI-driven Python SDK and CLI for skill lifecycle management; Supports skill monetization via Stripe and TON blockchain integration; Automated dependency resolution and local installation of agent skills Registry of reusable AI skills based on AgentSkills specification
