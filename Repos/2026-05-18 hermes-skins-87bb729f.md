# hermes-skins

Source: hermes-atlas
Category: Developer Tools
Author: joeynyc

Hermes Skins is a collection of community-contributed visual themes for the Hermes CLI agent. It allows users to customize the terminal interface by modifying YAML configuration files that control banner colors, ASCII art, spinner animations, and branding text. The project separates visual presentation from agent logic, enabling aesthetic transformations ranging from retro CRT styles to modern pastel palettes. Users can apply these themes globally or for specific sessions without altering the underlying AI behavior. Highlights: Customizes terminal UI elements including banners, spinners, and ASCII art; Uses YAML-based configuration for easy theme creation and sharing; Inherits from default settings to support partial skin definitions Community CLI skins and themes for Hermes terminal UI
