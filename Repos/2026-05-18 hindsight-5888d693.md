# hindsight

Source: hermes-atlas
Category: Memory & Context
Author: vectorize-io

Hindsight is an agent memory system designed to enable long-term learning and information retention for AI agents. It utilizes a workflow of retain, recall, and reflect operations to move beyond simple conversation history into persistent mental models. The system can be integrated via an LLM wrapper for automatic memory management or through a direct API for granular control over storage and retrieval. It supports multi-user isolation through metadata filtering and is compatible with major providers including OpenAI, Anthropic, and Ollama. Highlights: Implements retain, recall, and reflect workflows for long-term learning.; Provides an LLM wrapper for two-line code integration.; Supports PostgreSQL and Oracle AI Database for enterprise storage.
