# swarmclaw

Source: hermes-atlas
Category: Multi-Agent & Orchestration
Author: swarmclawai

SwarmClaw is a self-hosted AI agent runtime and multi-agent framework designed for building and managing autonomous agent swarms. It operates through an orchestration layer that supports durable memory, delegation, and scheduled tasks across more than 23 LLM providers. The platform features an organizational chart interface for visualizing agent activity and includes a conversation-to-skill learning system to expand agent capabilities. It integrates with various tools including MCP, OpenClaw gateways, and delegated CLI backends like Claude Code and Ollama. Highlights: Supports 23+ LLM providers including Hermes, OpenAI, and local Ollama.; Features durable agent memory and reviewed conversation-to-skill learning.; Orchestrates multi-agent swarms with delegation, heartbeats, and task scheduling. Build autonomous AI agent swarms with orchestration, skills, and multiple model providers
