# keep

Source: hermes-atlas
Category: Memory & Context
Author: keepnotes-ai

Keep is a reflective memory system for AI agents that enables long-term learning and contextual awareness. It processes diverse inputs—including URLs, codebases, and media files—by summarizing, embedding, and organizing them into a navigable graph of "edge tags." The system automatically surfaces relevant past learnings and commitments during agent sessions to improve decision-making. It supports fully local operation via Ollama or MLX, as well as cloud-hosted and API-based configurations.
