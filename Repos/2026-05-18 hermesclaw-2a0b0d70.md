# hermesclaw

Source: hermes-atlas
Category: Deployment & Infra
Author: TheAiSingularity

HermesClaw is a deployment framework that runs the Hermes Agent within NVIDIA’s OpenShell sandbox to provide hardware-level security enforcement. It utilizes kernel-level mechanisms like Landlock LSM and Seccomp BPF to restrict the agent's filesystem access, network egress, and system calls. Users can toggle between strict and permissive security policies to enable features like web searching or messaging integrations while maintaining an isolated environment. The project includes a library of over 40 built-in tools and skills for tasks such as automated research, code review, and multi-agent delegation. Highlights: Hardware-enforced sandboxing via NVIDIA OpenShell kernel-level security; Multi-layer protection blocking unauthorized network egress and filesystem writes; Pre-configured policy presets for strict, gateway, or permissive access Hermes Agent sandboxed with hardware-level enforcement
