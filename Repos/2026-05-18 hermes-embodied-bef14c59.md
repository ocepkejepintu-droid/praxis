# hermes-embodied

Source: hermes-atlas
Category: Domain Applications
Author: bryercowan

Hermes Embodied is a robotics extension for Hermes Agent that enables autonomous fine-tuning of Vision-Language-Action (VLA) models through natural language. The system orchestrates a self-improvement loop by collecting robot trajectories in simulation or hardware, provisioning cloud GPUs via Vast.ai, and retraining models when performance thresholds are met. It integrates with the LeRobot framework and MuJoCo simulation to automate the transition from data collection to model promotion. This allows users to manage complex robotics training pipelines without manual machine learning expertise. Highlights: Automates VLA fine-tuning loops using natural language commands; Provisions and manages Vast.ai cloud GPU instances autonomously; Supports SmolVLA and GR00T models for simulation or physical hardware Self-improving robotics via VLA model fine-tuning on cloud GPUs
