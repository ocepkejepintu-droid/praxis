# evey-bridge-plugin

Source: hermes-atlas
Category: Plugins & Extensions
Author: 42-evey

The Evey Bridge Plugin is a technical integration that connects Claude Code with the Evey hermes-agent stack. It functions by automatically checking for messages or tasks from the bridge and injecting them as context into every prompt submission. The plugin includes specialized commands for running a 'Mother Mode' improvement loop and monitoring system health, including wallet and cron status. It is designed specifically for users running the Evey stack at a designated local directory. Highlights: Injects Evey messages and tasks into Claude Code prompts; Automates Mother Mode improvement loops via the /evey-loop command; Monitors health, wallets, and crons with quick status checks Claude Code and Hermes context sharing bridge — auto-checks messages, Mother Mode loop
