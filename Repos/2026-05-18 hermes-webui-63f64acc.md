# hermes-webui

Source: hermes-atlas
Category: Workspaces & GUIs
Author: nesquena

Hermes WebUI provides a lightweight, browser-based interface for the Hermes Agent, offering a graphical alternative to the standard command-line experience. Built with Python and vanilla JavaScript, it maintains full parity with the CLI without requiring complex build steps or frameworks. The interface features a three-panel layout for managing chat sessions, workspace files, and agent settings through a centralized control center. It allows users to interact with persistent memory, scheduled cron jobs, and self-improving skills while remaining provider-agnostic across various AI models.
