# hermes-ui

Source: hermes-atlas
Category: Workspaces & GUIs
Author: pyrate-llama

Hermes UI is a glassmorphic web dashboard designed to serve as the centralized command center for the Hermes Agent ecosystem. It functions as a single-file React application served via a lightweight Python proxy, enabling users to interact with their self-hosted assistant without complex frontend deployments. The interface provides comprehensive visibility into agent operations through real-time log streaming, a task board for tracking active tool work, and integrated file browsing. It also features dedicated management views for MCP tools, cron jobs, and skill libraries to streamline agent orchestration.
