# hermes-tweet

Source: hermes-atlas
Category: Plugins & Extensions
Author: Xquik-dev

Hermes Tweet is a native plugin for the Hermes Agent ecosystem that enables automated interaction with X (formerly Twitter) via the Xquik API. It integrates 99 generated endpoints into Hermes as structured tools, allowing agents to perform tasks like searching, reading account data, and posting content. The plugin utilizes a least-privilege security model by separating read-only operations from account actions and requiring explicit environment configuration for write access. It includes a bundled skill for agent guidance and slash commands for quick status and trend checks.
