# gbrain

Source: hermes-atlas
Category: Memory & Context
Author: garrytan

GBrain is an opinionated memory and context layer designed to provide AI agents with a persistent, self-wiring knowledge graph. It utilizes a hybrid search approach that combines vector retrieval with structured entity linking to extract relationships like investments, employment, and founder roles without additional LLM calls. The system automates memory consolidation by ingesting diverse data sources such as meetings, emails, and tweets while performing overnight citation fixes. It includes specialized tools for tracking company trajectories and founder scorecards, making it suitable for high-volume information synthesis in production environments.
