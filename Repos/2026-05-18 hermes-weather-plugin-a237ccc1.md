# hermes-weather-plugin

Source: hermes-atlas
Category: Plugins & Extensions
Author: FahrenheitResearch

The Hermes Weather Plugin provides a suite of 13 meteorological tools for the Hermes Agent ecosystem, enabling real-time access to observations, forecasts, and high-fidelity model imagery. It utilizes a Python-first architecture that offloads heavy computational tasks, such as NEXRAD radar rendering and ECAPE calculations, to high-performance Rust backends. The plugin features an automated bootstrap process that compiles necessary Rust binaries on first use, ensuring a streamlined setup for complex weather analysis. Users can retrieve verified model soundings, severe weather outlooks from the SPC, and global data via Open-Meteo integration. Highlights: Thirteen tools for NWS forecasts, radar imagery, and severe weather alerts; Rust-backed meteorological calculations for ECAPE, soundings, and parcel profiles; Automated bootstrap system for compiling and caching required Rust binaries NWS-grade model imagery, NEXRAD radar, and verified meteorological calculations. Pure Rust.
