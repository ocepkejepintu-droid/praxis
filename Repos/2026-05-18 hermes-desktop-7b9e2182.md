# hermes-desktop

Source: hermes-atlas
Category: Workspaces & GUIs
Author: fathah

Hermes Desktop is a native GUI application for managing the installation, configuration, and operation of Hermes Agent. It automates the setup of local dependencies like Python and uv or connects to remote API servers to provide a streamlined alternative to CLI management. The app features a streaming chat interface, session history with full-text search, and a visual editor for personas and scheduled tasks. It integrates with 16 messaging gateways and 14 toolsets, including web browsing, code execution, and memory systems. Highlights: Guided installer for local Hermes Agent or remote API connection; Supports 16 messaging gateways including Telegram, Discord, and Slack; Built-in management for 14 toolsets, memory systems, and personas Desktop companion app for installing, configuring, and chatting with Hermes Agent
