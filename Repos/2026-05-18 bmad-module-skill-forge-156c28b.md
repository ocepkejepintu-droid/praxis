# bmad-module-skill-forge

Source: hermes-atlas
Category: Skills & Skill Registries
Author: armelhbobdad

Skill Forge is a development tool that converts source code, documentation, and developer discourse into verified instruction files for AI agents. It uses Abstract Syntax Tree (AST) analysis to extract precise function signatures and types, ensuring agents use libraries correctly without hallucinating API calls. Every generated instruction includes a provenance map linking back to specific file lines at pinned commits or documentation URLs. The project follows the agentskills.io specification, making the output compatible with AI tools like Cursor, Claude, and Copilot. Highlights: Generates version-pinned skills from codebases and documentation; Provides verifiable audit trails with file-and-line level provenance; Automates skill lifecycle management including updates and transactional exports Converts repos, docs, and developer discourse into agentskills.io-compliant skills
