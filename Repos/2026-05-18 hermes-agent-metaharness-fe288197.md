# hermes-agent-metaharness

Source: hermes-atlas
Category: Multi-Agent & Orchestration
Author: howdymary

Hermes Agent Meta-Harness is a standalone outer-loop optimization framework designed to improve the performance of Hermes agents on verifiable coding benchmarks. It treats the agent as an execution backend and searches for optimal harness configurations—the code governing context collection and retrieval—rather than modifying model weights. The system automates benchmark orchestration, archive analysis, and frontier tracking to identify high-performing harness candidates. It currently supports TBLite and TB2 benchmarks, providing structured mutation and search capabilities to refine agentic workflows. Highlights: Optimizes harness code through an automated outer-loop search process; Orchestrates TBLite and TB2 benchmark evaluation via Hermes Agent; Tracks performance frontiers with structured candidate mutation and comparison Meta-harness for Hermes Agent — meta-optimization with arxiv paper reference
