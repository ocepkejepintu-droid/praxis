# hermes-telegram-miniapp

Source: hermes-atlas
Category: Workspaces & GUIs
Author: clawvader-tech

Hermes Telegram Mini App is a terminal-style web interface that allows users to interact with their Hermes agent directly within Telegram. It functions as a React SPA that connects to the Hermes Agent via a secure gateway, utilizing dual HMAC-SHA256 and Ed25519 validation for authentication. The interface provides a mobile-first dashboard for managing agent instances, monitoring system health, and handling scheduled cron jobs. It also supports file attachments with automatic vision analysis and OCR capabilities.
