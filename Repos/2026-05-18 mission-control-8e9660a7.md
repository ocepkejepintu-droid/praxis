# mission-control

Source: hermes-atlas
Category: Multi-Agent & Orchestration
Author: builderz-labs

Mission Control is a self-hosted dashboard designed to orchestrate and monitor fleets of AI agents. It utilizes a local-first architecture powered by SQLite and Next.js to coordinate multi-agent workflows, dispatch tasks, and track operational costs without external database dependencies. The platform features built-in quality gates for task approval, real-time telemetry via WebSockets, and a security framework that includes secret detection and trust scoring. It integrates with various agent frameworks like CrewAI and LangGraph, and provides a bridge for CLI tools such as Claude Code. Highlights: Self-hosted orchestration using SQLite with zero external dependencies; Multi-gateway support for CrewAI, LangGraph, AutoGen, and Claude SDK; Built-in Aegis review system for human-in-the-loop task sign-off Self-hosted AI agent orchestration — dispatch tasks, run multi-agent workflows, monitor spend
