# hermes-android

Source: hermes-atlas
Category: Integrations & Bridges
Author: raulvidis

hermes-android is a remote control system that enables AI agents to interact with Android devices. It utilizes a Kotlin-based bridge app that connects to a Python relay server via WebSockets, allowing for control behind NATs without VPNs or port forwarding. The system provides 36 specialized tools for UI automation, device management, and hardware access. It is compatible with both standard Android phones and Android Automotive OS head units. Highlights: 36 tools for UI interaction, screen recording, and device control; WebSocket-based connection using a 6-character pairing code; Support for Android Automotive OS car head units Android device control — bridge app + Python toolset for mobile automation
