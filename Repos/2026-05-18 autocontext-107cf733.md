# autocontext

Source: hermes-atlas
Category: Memory & Context
Author: greyhaven-ai

Autocontext is a recursive self-improving harness that enables agents to succeed on complex tasks through iterative refinement. It operates by executing a goal against real-world evaluations, retaining successful strategies while discarding failures to produce structured playbooks and artifacts. The system uses five specialized roles—competitor, analyst, coach, architect, and curator—to manage knowledge persistence and tool generation. It integrates with various providers including Anthropic, OpenAI, and local runtimes like Pi, exporting structured traces and playbooks for future agent iterations. Highlights: Iterative self-improvement through structured evaluation and knowledge persistence; Five-role architecture manages strategy, analysis, coaching, and curation; Produces reusable playbooks, datasets, and detailed execution traces Recursive self-improving context harness — helps agents succeed on complex tasks
