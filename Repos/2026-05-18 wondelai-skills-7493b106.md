# wondelai/skills

Source: hermes-atlas
Category: Skills & Skill Registries
Author: wondelai

Wondel.ai Skills is a cross-platform library of agent capabilities designed for Claude Code and agentskills.io-compatible environments. It provides pre-configured skill collections based on established industry frameworks, such as Jobs to Be Done, Clean Code, and Lean Startup methodology. Users can install these skills via the Claude Code marketplace or a dedicated CLI to equip AI agents with specialized knowledge in software architecture, UX design, and product strategy. The project bridges the gap between raw LLM capabilities and structured professional methodologies through modular, installable plugins. Highlights: Compatible with Claude Code and agentskills.io-compatible agents; Modular installation via marketplace or npx skills CLI; Frameworks for architecture, UX design, marketing, and product strategy Cross-platform skills library for Claude Code and agentskills.io-compatible agents
