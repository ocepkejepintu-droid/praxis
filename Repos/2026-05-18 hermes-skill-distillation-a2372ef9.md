# hermes-skill-distillation

Source: hermes-atlas
Category: Forks & Derivatives
Author: beardthelion

This project provides a skill distillation pipeline designed to fine-tune Hermes 4 models using real-world agentic trajectories. It captures tool-use sessions from a diverse 30-task battery and converts them into SFT-ready data or live RL signals for the Atropos training framework. The system uses a multi-dimensional reward function to score trajectories based on task completion, efficiency, and error recovery. This approach enables a closed learning loop where production-style agent interactions serve as the primary training data for future model iterations. Highlights: Generates SFT-ready JSONL data from real-world tool usage trajectories; Integrates with Atropos for live reinforcement learning via GRPO; Automated scoring based on completion, efficiency, and error recovery Generate agentic training trajectories from real-world tasks for Hermes 4 fine-tuning
