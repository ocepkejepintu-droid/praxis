# opencode-hermes-multiagent

Source: hermes-atlas
Category: Multi-Agent & Orchestration
Author: 1ilkhamov

Hermes is a multi-agent orchestration system for OpenCode AI designed to automate the full software development lifecycle. It utilizes a master orchestrator to route tasks through specialized pipelines involving 17 distinct agents focused on research, planning, implementation, quality, and infrastructure. The system enforces mandatory quality gates and security audits while managing context flow between models like GPT-5.2, Gemini, and Claude. It supports complex workflows such as security-sensitive feature development, automated bug investigation, and performance optimization. Highlights: Coordinates 17 specialized agents through role-specific development pipelines; Enforces mandatory reviewer and tester gates for all code changes; Integrates with MCP servers for external documentation and web fetching 17 specialized agents for research, planning, implementation, quality, and infrastructure
