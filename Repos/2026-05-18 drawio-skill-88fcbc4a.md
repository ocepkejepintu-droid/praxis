# drawio-skill

Source: hermes-atlas
Category: Skills & Skill Registries
Author: Agents365-ai

This project is a diagramming skill that converts natural-language descriptions into professional .drawio XML files and high-resolution image exports. It utilizes the native draw.io desktop CLI to render diagrams and features an automated self-check mechanism that reads its own output to fix layout issues like overlapping labels or stacked edges. The tool includes six specialized presets for architecture, UML, and machine learning diagrams while supporting iterative refinement through a multi-round feedback loop. It integrates directly with agent environments such as Claude Code, Cursor, and Hermes without requiring a background daemon. Highlights: Generates .drawio XML and exports to PNG, SVG, PDF, or JPG; Auto-fixes layout issues using a self-check visual feedback loop; Includes six diagram presets for architecture, UML, and ML models Generate draw.io diagrams from natural language descriptions
