# bigiron

Source: hermes-atlas
Category: Multi-Agent & Orchestration
Author: supermodeltools

Big Iron is an experimental AI-native software development system that automates the full development lifecycle through a graph-gated workflow. It integrates Hermes Agent with the Supermodel Code Graph API to replace traditional file-loading context with structural intelligence, including call graphs and dependency maps. The system executes an eight-phase process—from planning to autonomous health checks—using specific skills and quality gates to ensure architectural alignment. While now deprecated in favor of the Supermodel CLI, it serves as a prototype for autonomous, graph-grounded code generation and refactoring. Highlights: Executes an autonomous eight-phase SDLC from planning to health checks; Uses Supermodel MCP to provide Hermes with structural code graphs; Enforces architectural quality through automated graph-diff phase gates AI-native SDLC — Hermes Agent + Supermodel code graph, graph-gated at every phase
