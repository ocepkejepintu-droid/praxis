# hermes-web-search-plus

Source: hermes-atlas
Category: Plugins & Extensions
Author: robbyczgw-cla

web-search-plus is a capability-based plugin for Hermes Agent that provides multi-provider web search and URL extraction. It utilizes a Routing v2 engine to automatically select from available search providers based on the user's configured API keys. The plugin features additive extraction capabilities and includes cost-control mechanisms like research mode to manage provider usage. Highlights: Auto-routes queries across multiple search providers; Provides clean URL extraction via specialized backends; Configurable priority and fallback provider settings Multi-provider web search (Serper, Tavily, Exa, Querit, Perplexity) with auto-routing
