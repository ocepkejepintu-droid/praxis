# hermes-webui

Source: hermes-atlas
Category: Workspaces & GUIs
Author: sanchomuzax

Hermes WebUI is a dedicated process monitoring and configuration dashboard for the Hermes Agent ecosystem. It functions as a non-intrusive polling bridge that reads from the agent's SQLite database and YAML files to provide real-time updates without requiring modifications to the core agent code. Users can track session histories, monitor platform health and costs, and manage scheduled cron jobs through a responsive, themeable interface. The system also includes a secure, read-only configuration viewer and a skill browser for inspecting built-in and custom agent capabilities. Highlights: Real-time dashboard for session activity, gateway status, and cost tracking; Full conversation history inspection with FTS5-powered search functionality; Read-only management of config.yaml, environment variables, and cron jobs Process monitoring and configuration dashboard for Hermes
