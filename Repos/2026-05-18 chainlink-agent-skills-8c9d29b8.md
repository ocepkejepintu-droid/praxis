# chainlink-agent-skills

Source: hermes-atlas
Category: Skills & Skill Registries
Author: smartcontractkit

Chainlink Agent Skills is a collection of coding skills designed to enable AI agents to interact with oracle networks and smart contracts. The project implements the agentskills.io specification to provide standardized capabilities for agent discovery and activation. It includes specialized skills for managing CCIP, Data Feeds, VRF, ACE, Data Streams, and CRE operations. These tools allow agents to generate workflows, handle on-chain verification, and build cross-chain applications. Highlights: Follows the agentskills.io specification for agent compatibility; Supports CCIP, VRF, Data Feeds, and Data Streams; Includes CRE onboarding and runtime operation capabilities Oracle network and smart contract interaction skills (agentskills.io spec)
