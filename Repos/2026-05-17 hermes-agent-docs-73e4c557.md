# hermes-agent-docs

Source: hermes-atlas
Category: Guides & Docs
Author: mudrii

Hermes Agent is an autonomous AI system designed to create a persistent, self-improving learning loop across multiple platforms. It utilizes a unified core to manage long-running conversations, execute parallel tool calls, and autonomously generate reusable skills stored in a local SQLite database. The agent integrates with over 20 messaging platforms and dozens of model providers, allowing it to function as a CLI, a messaging gateway, or an IDE extension. Its architecture supports context compression, full-text session search, and a plugin system for extending tools and backends without forking the codebase. Highlights: Autonomous skill creation and self-improvement based on task experience; Unified messaging gateway supporting Telegram, Discord, Slack, and WhatsApp; Pluggable transport layer compatible with 400+ models and custom endpoints Community documentation covering deployment patterns and v0.2.0+ workflows
