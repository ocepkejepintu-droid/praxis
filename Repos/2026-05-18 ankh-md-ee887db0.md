# Ankh.md

Source: hermes-atlas
Category: Developer Tools
Author: Abruptive

Ankh.md is an open-source framework designed to transform a single Hermes Agent into multiple specialized, project-specific instances. It works by detecting a local `.agent/` directory and dynamically overriding the default `hermes` command with a scoped agent configuration. This allows developers to maintain unique instructions, skills, and memories for different codebases or workflows without cross-project interference. Highlights: Per-folder scoping with local configuration and skills; Seamlessly switches between default and scoped Hermes agents; Supports project-local memories, tools, and custom instructions TAW Agent framework for creating scoped Hermes Agents
