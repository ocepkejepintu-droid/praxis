# mnemosyne

Source: hermes-atlas
Category: Memory & Context
Author: AxDSan

Mnemosyne is a local-first memory system designed for the Hermes Agent framework to provide private, zero-dependency context management. It utilizes SQLite with native vector search and full-text search to store conversations and knowledge without requiring external databases or API keys. The system operates entirely in-process to eliminate network latency, achieving sub-millisecond read and write speeds. It features a three-tier architecture comprising working, episodic, and scratchpad memory to handle long-term information retention.
