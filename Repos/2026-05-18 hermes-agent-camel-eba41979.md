# hermes-agent-camel

Source: hermes-atlas
Category: Forks & Derivatives
Author: nativ3ai

This project is a version of the Hermes Agent that integrates CaMeL-style trust boundaries to protect against indirect prompt injection. It implements an opt-in runtime guard that monitors or enforces security policies by gating sensitive tool side effects against verified user intent. The system maintains the core Hermes features, including a self-improving learning loop, autonomous skill creation, and cross-platform accessibility via Telegram, Discord, and Slack. Users can toggle between monitoring modes to record policy decisions or enforcement modes to block unauthorized actions during autonomous execution. Highlights: Opt-in CaMeL runtime guard for indirect prompt-injection protection; Self-improving learning loop with autonomous skill creation and memory; Multi-platform gateway supporting Telegram, Discord, Slack, and Signal Hermes with integrated CaMeL trust boundaries for safer autonomous execution
