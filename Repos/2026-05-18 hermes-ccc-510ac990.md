# hermes-CCC

Source: hermes-atlas
Category: Developer Tools
Author: AlexAI-MCP

hermes-CCC is a port of the Hermes Agent framework specifically designed for the Claude Code native skill system. It integrates the Hermes brain architecture directly into the Claude Code execution context, eliminating the need for external processes or OAuth authentication. The project implements 46 core skills across categories such as MLOps, software development, and research, including specialized tools for RL training and vector database management. This allows developers to utilize Hermes-style memory, personas, and task routing while remaining within a unified terminal environment. Highlights: Port of Hermes Agent brain and 46 core skills to CCC; Runs natively in Claude Code without separate processes or OAuth; Includes MLOps tools for vLLM, GGUF inference, and RL training Hermes Agent ported to Claude Code Channel — 46 native skills, no OAuth, no external process
