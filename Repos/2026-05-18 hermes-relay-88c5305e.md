# hermes-relay

Source: hermes-atlas
Category: Integrations & Bridges
Author: Codename-11

Hermes-Relay is a companion suite that enables remote interaction with a server-deployed Hermes agent through a native Android app and a desktop CLI. It uses a WebSocket Secure (WSS) relay to bridge local devices with remote agent instances, allowing for persistent sessions across different platforms. The Android application provides mobile device control including screen reading and gesture execution, while the desktop client allows a remote agent to execute local terminal commands and file operations. This setup eliminates the need for SSH by routing tool execution and chat interfaces through a unified relay plugin.
