# hermes-control-interface

Source: hermes-atlas
Category: Workspaces & GUIs
Author: xaspx

The Hermes Control Interface is a self-hosted web dashboard designed to manage the Hermes AI agent stack through a centralized graphical interface. It utilizes a Node.js and Express backend with WebSockets to provide real-time terminal access, multi-agent gateway management, and live log streaming. Users can configure over 80 agent settings, monitor token analytics and costs, and manage scheduled cron jobs across multiple profiles. The platform also includes a secure file explorer, a revamped chat interface with tool-call visualization, and robust role-based access control. Highlights: Multi-agent gateway for starting, stopping, and configuring individual profiles.; Real-time chat interface with collapsible tool-call cards and session history.; Comprehensive token analytics tracking usage, costs, and model performance.
