# skillsdotnet

Source: hermes-atlas
Category: Skills & Skill Registries
Author: PederHP

SkillsDotNet is a C#/.NET implementation of the agentskills.io standard designed to integrate agent skills into the Model Context Protocol (MCP). It uses a skill:// URI convention to implement progressive disclosure, allowing clients to discover lightweight skill metadata before loading full documentation or supporting files. The library provides seamless integration with Microsoft.Extensions.AI and supports automatic discovery of skills from various agent directories like Claude, Cursor, and Copilot. Highlights: Implements progressive disclosure via SKILL.md files; Integrates with Microsoft.Extensions.AI for tool-based skill loading; Supports automatic registration of common agent skill directories C# / .NET implementation of agentskills.io with MCP integration
