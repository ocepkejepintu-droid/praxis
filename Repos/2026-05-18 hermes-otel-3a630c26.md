# hermes-otel

Source: hermes-atlas
Category: Plugins & Extensions
Author: briancaffey

hermes-otel is an OpenTelemetry plugin designed to provide observability for the Hermes Agent ecosystem. It automatically captures and exports LLM tool calls, model invocations, and API requests as OTel spans to compatible backends. The plugin supports single or multi-backend configurations, allowing users to stream telemetry to various tracing and monitoring platforms simultaneously.
