# hermes-agent

Source: hermes-atlas
Category: Core & Official
Author: NousResearch

Hermes Agent is an autonomous AI system designed for persistent, cross-platform task execution and self-improvement. It utilizes a closed learning loop to generate new skills from experience and build long-term user models through memory nudges and session searches. The agent integrates with 14 messaging platforms and supports six execution backends, allowing it to run on everything from local hardware to serverless cloud environments. Users can automate complex workflows via a built-in cron scheduler and delegate tasks to parallel subagents. Highlights: Autonomous skill creation and self-improvement from task experience; Multi-platform support including Telegram, Discord, Slack, and CLI; Flexible execution across local, Docker, SSH, and serverless backends The self-improving AI agent that grows with you — persistent memory, auto-generated skills, 14 platforms, 6 execution backends
