Source: hermes-atlas

The Hermes Agent Guide is a comprehensive Chinese-language documentation project designed to bridge the gap between fragmented information and practical implementation of the Hermes Agent framework. It provides a structured 16-book knowledge map covering the system's five-layer architecture, three-tier memory systems, and the integration of over 47 built-in tools. The guide details technical workflows for local and cloud deployment, MCP protocol automation, and multi-platform connectivity with services like WeChat and Discord. Users can follow specialized learning paths for development, commercial monetization, or migrating from the OpenClaw framework.
