# anihermes

Source: hermes-atlas
Category: Domain Applications
Author: rodmarkun

AniHermes is a natural language anime media server and tracker built on the Hermes Agent framework. It leverages LLM orchestration to manage downloads via qBittorrent, track series progress, and organize local libraries through conversational interfaces like Telegram, Discord, or CLI. The system utilizes agentic capabilities such as vision analysis for identifying anime from screenshots and automated CRON jobs for scheduling weekly episode downloads. It integrates directly with Anilist for watchlist synchronization and provides parallel search functionality across multiple sources like SubsPlease and Nyaa. Highlights: Natural language control for downloads, tracking, and library management; Vision-based anime identification from screenshots using Hermes Agent; Automated weekly episode tracking via NL-defined CRON jobs Local anime server and tracker via natural language for Hermes Agent
