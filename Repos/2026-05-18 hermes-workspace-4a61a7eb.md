# hermes-workspace

Source: hermes-atlas
Category: Workspaces & GUIs
Author: outsourc-e

Hermes Workspace is a native web-based command center designed to provide a unified interface for managing Hermes Agents. It integrates chat, a cross-platform terminal, and a file browser with a memory editor and skills manager to streamline agent orchestration. The platform features a Swarm Mode for role-based dispatching and persistent tmux-backed workers, allowing users to monitor multi-agent operations through a centralized dashboard. It maintains zero-fork parity with the official Nous Research Hermes Agent, connecting via API to existing gateway and dashboard services. Highlights: Unified interface for chat, terminal, memory browsing, and skill management; Swarm Mode for autonomous multi-agent orchestration and role-based dispatching; Zero-fork integration with vanilla Nous Research Hermes Agent installations Native web workspace — chat, terminal, memory browser, skills manager, and inspector panel
