# MeiGen-AI-Design-MCP

Source: hermes-atlas
Category: Plugins & Extensions
Author: jau123

MeiGen AI Design MCP is an open-source Model Context Protocol server that integrates professional image and video generation directly into AI coding environments. It connects to nine leading models, including Midjourney and Flux, via three backend modes: the MeiGen cloud, OpenAI-compatible APIs, or local ComfyUI instances. The system features a library of over 1,400 curated prompt templates and supports parallel sub-agent orchestration for batch processing. It is designed for use in tools like Hermes Agent, Claude Code, and Cursor to facilitate design tasks like logo creation and product rendering.
