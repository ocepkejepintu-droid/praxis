# atropos

Source: hermes-atlas
Category: Core & Official
Author: NousResearch

Atropos is an environment microservice framework designed for asynchronous reinforcement learning (RL) with large language models. It operates by setting up environments as services and utilizing a trajectory API to manage data flow between the environment, trainer, and inference engine. The system supports a wide range of interactive settings, including code execution, multi-turn tool calling, and multimodal tasks. By standardizing how models interact with external tools and datasets, it enables the training of specialized models for complex reasoning and functional tasks. Highlights: Microservice framework for asynchronous RL environment management; Trajectory API for scalable data collection and evaluation; Supports tool calling, code execution, and multimodal environments RL training environments framework for tool-calling models
