# hermes-council

Source: hermes-atlas
Category: Integrations & Bridges
Author: Ridwannurudeen

Hermes Council is an MCP server that provides an adversarial judgment layer for Hermes Agent to stress-test plans and actions before execution. It utilizes a multi-perspective deliberation process involving personas like Advocate, Skeptic, and Arbiter to generate structured verdicts on code diffs, claims, and risky commands. This approach helps prevent overconfident agent failures by producing explicit signals such as blocking risks, required checks, and DPO preference pairs for reinforcement learning workflows. Highlights: Multi-persona adversarial deliberation using Advocate, Skeptic, and Oracle roles; Structured safety gating with allow, deny, and conditional verdicts; Generates DPO preference pairs and rewards for RL training workflows Adversarial multi-perspective council MCP server for decision-making
