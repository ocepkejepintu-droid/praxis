# hermes-paperclip-adapter

Source: hermes-atlas
Category: Core & Official
Author: NousResearch

The hermes-paperclip-adapter is a bridge that allows Hermes Agent to function as a managed employee within the Paperclip company ecosystem. It works by wrapping the Hermes CLI to parse raw terminal output into structured transcript entries, enabling Paperclip to render interactive tool cards and status icons. The adapter manages session persistence across heartbeats, allowing agents to maintain context while responding to task assignments or issue comments. It also synchronizes skills between local directories and the Paperclip interface while reclassifying system logs to ensure a clean user experience. Highlights: Integrates 30+ native tools and 80+ skills into Paperclip; Parses raw CLI output into structured markdown and tool cards; Supports eight inference providers including Anthropic, OpenAI, and OpenRouter Run Hermes as a managed employee in Paperclip company systems
