# Anthropic-Cybersecurity-Skills

Source: hermes-atlas
Category: Skills & Skill Registries
Author: mukul975

Anthropic Cybersecurity Skills is an open-source library of 754 structured skills designed to provide AI agents with the domain-specific workflows of a senior security analyst. Each skill follows the agentskills.io standard, utilizing YAML frontmatter and structured Markdown to guide agents through step-by-step execution and verification. The repository maps every skill across five major frameworks, including MITRE ATT&CK, NIST CSF 2.0, and the NIST AI RMF. It supports over 26 security domains such as cloud security and threat hunting, integrating directly with platforms like Claude Code and Hermes Agent. Highlights: 754 structured cybersecurity skills across 26 specialized security domains; Unified mapping to MITRE ATT&CK, ATLAS, D3FEND, and NIST frameworks; Compatible with agentskills.io standard for sub-second AI skill discovery 754 structured cybersecurity skills mapped to MITRE ATT&CK, NIST CSF 2.0, ATLAS, D3FEND & NIST AI RMF
