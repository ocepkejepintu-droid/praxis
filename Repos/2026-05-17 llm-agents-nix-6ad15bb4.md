# llm-agents.nix

Source: hermes-atlas
Category: Deployment & Infra
Author: numtide

llm-agents.nix is a collection of Nix packages designed to simplify the deployment and management of AI coding agents and development tools. The project provides a centralized repository that automatically updates daily, ensuring users have access to the latest versions of various agentic tools. It allows developers to run a wide range of CLI-based agents, such as Goose, Claude Code, and Bernstein, directly via Nix flakes without manual installation. This infrastructure supports multi-agent orchestration, unified API proxies, and terminal-based AI pair programming across different model providers. Highlights: Provides daily updated Nix packages for numerous AI coding agents; Enables instant execution of tools via Nix run commands; Supports diverse agents including local, open-source, and proprietary binaries Nix packages for AI coding agents including Hermes
