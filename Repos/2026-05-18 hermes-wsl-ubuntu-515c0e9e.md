# hermes-wsl-ubuntu

Source: hermes-atlas
Category: Guides & Docs
Author: metantonio

This project provides a comprehensive setup guide and automation scripts for running Hermes Agent locally on Windows via WSL2 or natively on macOS. It streamlines the integration of llama.cpp, the Qwen3.5 model family, and the Camofox browser server to enable local agentic workflows with GPU acceleration. The repository includes an automated installation script that configures the environment, builds necessary backends like CUDA or Metal, and sets up the Hermes HUDUI web interface. By focusing on local execution, it allows users to maintain data privacy while utilizing browser-automation and LLM capabilities. Highlights: Automated installation scripts for WSL2 Ubuntu and macOS environments; Supports CUDA and Metal acceleration for local LLM inference; Integrates Camofox browser server for agentic web-browsing tasks Step-by-step WSL2 Ubuntu setup guide for Windows users
