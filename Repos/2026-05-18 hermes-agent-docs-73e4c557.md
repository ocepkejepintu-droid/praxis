# hermes-agent-docs

Source: hermes-atlas
Category: Guides & Docs
Author: mudrii

Hermes Agent is an autonomous AI system designed for persistent learning and multi-platform task execution. It utilizes a closed learning loop that creates and patches reusable skills while maintaining long-term memory through SQLite and FTS5 session search. The agent operates across diverse interfaces including a messaging gateway for over 20 platforms, an Ink-based TUI, and an OpenAI-compatible API server. It supports a wide range of model providers and features a plugin architecture for extending tools, transports, and backend capabilities. Highlights: Autonomous skill creation and self-improvement based on task experience; Unified messaging gateway supporting Telegram, Discord, Slack, and WhatsApp; Pluggable transport layer for native integration with 20+ model providers Community documentation covering deployment patterns and v0.2.0+ workflows
