# icarus-plugin

Source: hermes-atlas
Category: Skills & Skill Registries
Author: esaradev

Icarus is a Hermes plugin designed to create self-improving agents through shared memory and automated model fine-tuning. It captures agent decisions, session outcomes, and structured notes into a markdown-based "fabric" directory that can be optionally managed via Obsidian. This data is then used to extract high-quality training pairs for fine-tuning cheaper replacement models via Together AI. The system includes a full pipeline for evaluating candidate models against base versions and switching to them once performance thresholds are met. Highlights: Cross-instance shared memory via markdown-based fabric directories; Automated training data extraction with decision-quality tagging; End-to-end pipeline for fine-tuning and evaluating replacement models Self-memory and replacement models — remember your work, train your replacement
