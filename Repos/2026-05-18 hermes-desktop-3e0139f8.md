# hermes-desktop

Source: hermes-atlas
Category: Workspaces & GUIs
Author: dodo-reach

Hermes Desktop is a native macOS application that provides a localized workbench for managing Hermes Agent hosts over SSH. The app connects directly to remote hosts to manage sessions, Kanban boards, and files without requiring a browser wrapper or additional sync layers. Users can save local workflow presets to launch agent tasks in an embedded terminal while maintaining the remote host as the single source of truth. It supports multiple agent profiles and includes built-in tools for editing skills and monitoring resource usage. Highlights: Direct SSH connection to remote Hermes hosts without middleman APIs; Native macOS interface for sessions, Kanban, and remote file management; Local workflow presets for launching tasks in an embedded terminal A native Mac workspace for Hermes: real SSH, real terminal, real session data.
