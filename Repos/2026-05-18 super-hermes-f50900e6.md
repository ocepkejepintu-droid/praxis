# super-hermes

Source: hermes-atlas
Category: Skills & Skill Registries
Author: Cranot

Super Hermes is a skill registry for the Hermes Agent that enables models to generate their own analytical frameworks, or prisms, for deep code and document inspection. It works by instructing the agent to write specific thinking instructions before execution, allowing it to derive structural trade-offs and conservation laws rather than just identifying surface-level bugs. The system includes specialized commands like /prism-reflect, which provide a constraint report detailing what the analysis missed to ensure transparency in the agent's cognitive limits.
