# zouroboros-swarm-executors

Source: hermes-atlas
Category: Integrations & Bridges
Author: marlandoj

Zouroboros Swarm Executors is a bridge system that integrates local CLI-based AI tools into the Zo Computer swarm orchestration ecosystem. It functions by using bash bridge scripts to wrap tools like Hermes, Claude Code, and Gemini, allowing them to receive tasks via standard input and return results via standard output. This approach enables local execution of complex tasks such as security audits and web research while maintaining a shared identity and memory system across all personas. The project includes a health-check system to validate environment configurations and a registry for adding custom CLI tools as swarm members. Highlights: Wraps CLI tools into AI personas with shared identity files; Provides bridge scripts for Hermes, Claude Code, Gemini, and Codex; Routes tasks locally instead of through remote API calls Claude Code and Hermes task handoff bridge for zo-swarm-orchestrator
